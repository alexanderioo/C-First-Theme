# Клавиши — консольный тренажёр печати
Frolov Alexander 231-322

Учебный проект на C# и .NET 9. Приложение работает прямо в Terminal и подходит
для macOS, Linux и Windows.

Консольный формат выбран потому, что WPF поддерживается только на Windows.

## Возможности

- выбор словаря перед тренировкой;
- встроенные русские, английские и C#-словари;
- создание и редактирование собственных словарей;
- удаление пользовательских словарей;
- посимвольная цветовая подсветка ввода;
- исправление ошибок клавишей `Backspace`;
- прерывание тренировки клавишей `Esc`;
- расчёт скорости, точности, времени и количества ошибок;
- светлая и тёмная цветовые темы;
- настройка ширины текста, обратного отсчёта и живых показателей;
- сохранение словарей, настроек и статистики в JSON;
- общая статистика, история и результаты по словарям.

## Запуск на macOS

Откройте Terminal и выполните:

```bash
cd "/Users/frolovs/Desktop/C# 1"
dotnet run --project TypingTrainer
```

Остановка программы:

```text
Ctrl + C
```

## Размер шрифта

Консольная программа не управляет системным шрифтом Terminal. На Mac:

- `Cmd + Plus` — увеличить;
- `Cmd + Minus` — уменьшить;
- `Cmd + 0` — вернуть обычный размер.

## Расчёты

- скорость: `количество символов / время в минутах`;
- слова в минуту: `скорость в символах / 5`;
- точность: `(все нажатия - ошибки) / все нажатия × 100%`.

Исправленная через `Backspace` ошибка продолжает учитываться. Тренировка
завершается только после полного совпадения введённого и исходного текста.

## Хранение данных

На macOS JSON-файлы создаются в:

```text
~/Library/Application Support/TypingTrainer/
```

Файлы:

- `dictionaries.json` — словари;
- `settings.json` — настройки;
- `statistics.json` — результаты тренировок.

Путь можно переопределить переменной окружения `TYPING_TRAINER_DATA_DIR`.

## Структура проекта

```text
TypingTrainer/
├── ConsoleUI/
│   ├── ConsoleApplication.cs  # главное меню
│   ├── DictionaryMenu.cs      # выбор и редактор словарей
│   ├── TypingSession.cs       # обработка клавиш и тренировка
│   ├── SettingsMenu.cs        # настройки Terminal
│   └── StatisticsMenu.cs      # просмотр результатов
├── Models/                    # модели словарей, настроек и статистики
├── Services/                  # чтение и запись JSON
└── Program.cs                 # точка входа
```
