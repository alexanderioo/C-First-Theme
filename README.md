# Клавиши — тренажёр набора текста

Учебный веб-проект на C# и .NET 9 для развития скорости и точности печати.
Интерфейс и сценарий тренировки вдохновлены сервисами наподобие
[Клавогонок](https://klavogonki.ru/), но реализованы самостоятельно на Blazor.

## Возможности

- выбор словаря перед каждым заездом;
- встроенные словари для русского, английского, частых слов и C#;
- создание, редактирование и удаление пользовательских словарей;
- случайный выбор задания из текущего словаря;
- посимвольная подсветка правильного ввода и ошибок;
- обязательное исправление опечаток с помощью `Backspace`;
- расчёт скорости, точности, времени и количества ошибок;
- настройка размера и семейства шрифта, межстрочного интервала и темы;
- автоматическое сохранение настроек и результатов в JSON;
- дашборд с рекордами, графиком, статистикой по словарям и историей.

## Пользовательский сценарий

1. Пользователь открывает раздел «Словари».
2. Выбирает встроенный набор или создаёт собственный.
3. Нажимает «Начать заезд» и запускает таймер.
4. Набирает текст, исправляя подсвеченные ошибки.
5. Получает итоговую скорость и точность.
6. Результат автоматически появляется в разделе «Статистика».

## Запуск

Требуется [.NET 9 SDK](https://dotnet.microsoft.com/download/dotnet/9.0).

```bash
git clone git@github.com:alexanderioo/C-First-Theme.git
cd C-First-Theme
dotnet run --project TypingTrainer
```

После запуска приложение доступно по адресу:

```text
http://localhost:5178
```

Проверка сборки:

```bash
dotnet build TypingTrainer.sln --configuration Release
```

## Расчёты

- скорость: `количество символов / время в минутах`;
- слова в минуту: `скорость в символах / 5`;
- точность: `(нажатия - ошибки) / нажатия × 100%`.

Исправленная ошибка продолжает учитываться в результате. Заезд завершается
только после полного совпадения введённого и исходного текста.

## Хранение данных

Приложение использует локальные JSON-файлы:

- `App_Data/dictionaries.json`;
- `App_Data/settings.json`;
- `App_Data/statistics.json`.

Папка `App_Data` исключена из Git. При первом запуске встроенные словари
создаются автоматически, поэтому для работы проекта не нужна база данных.

## Структура

```text
TypingTrainer/
├── Components/
│   ├── Layout/       # общий каркас и навигация
│   └── Pages/        # тренировка, словари, настройки, статистика
├── Models/           # словари, результаты и настройки
├── Services/         # JSON-репозитории
├── wwwroot/          # глобальные стили и favicon
└── Program.cs        # регистрация сервисов и запуск приложения
```

Проект адаптирован для настольных и мобильных экранов.
