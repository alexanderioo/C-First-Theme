# Тренажёр набора текста

Консольный учебный проект на C# и .NET 9 для тренировки скорости и
точности печати.

## Возможности

- русский и английский языки;
- три уровня сложности;
- тренировка с собственным текстом;
- цветовая подсветка правильных и ошибочных символов;
- исправление с помощью `Backspace` и отмена с помощью `Esc`;
- расчёт времени, скорости, точности и количества ошибок;
- сохранение истории тренировок в JSON;
- общая статистика и десять последних результатов.

## Запуск

Из корневой папки проекта:

```bash
dotnet run --project TypingTrainer
```

Или сначала собрать решение:

```bash
dotnet build TypingTrainer.sln
dotnet run --project TypingTrainer
```

## Расчёты

- символов в минуту: `количество символов / время в минутах`;
- слов в минуту: `символов в минуту / 5`;
- точность: `(все нажатия - ошибки) / все нажатия * 100%`.

Исправленная через `Backspace` ошибка всё равно учитывается. Это позволяет
оценить не только итоговый текст, но и аккуратность набора.

## Структура

- `Program.cs` — точка входа;
- `TypingTrainerApplication.cs` — меню и сценарии программы;
- `TypingSession.cs` — обработка клавиш и проведение тренировки;
- `Services/TextProvider.cs` — встроенные учебные тексты;
- `Services/JsonResultRepository.cs` — сохранение результатов;
- `SessionResult.cs` и `StatisticsSummary.cs` — модели и расчёт статистики.

Файл результатов создаётся в локальной папке данных пользователя:
`TypingTrainer/results.json`.

Для разработки путь можно переопределить переменной окружения
`TYPING_TRAINER_DATA_FILE`.
